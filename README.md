# 二手交易管理系统 java68

## 项目概述

二手交易管理系统是一个基于B/S架构的Web应用程序，采用前后端分离的模式进行开发。本项目使用Java和Vue.js为主要技术栈，结合SpringBoot框架，实现了便捷的二手物品交易流程管理。

## 技术栈

- **后端：** Java、SpringBoot、Mybatis、Mysql
- **前端：** Vue.js
- **开发环境：** IDEA、JDK1.8、Maven、Mysql8.0

## 功能模块

### 前台功能

- **首页：**
  - 全部查询
  - 分类查询
  - 关键字搜索
- **发布物品：** 允许用户发布自己的二手物品
- **查看消息：** 查看系统通知和个人消息
- **个人中心：**
  - 编辑个人信息
  - 编辑收货地址
- **状态查询：**
  - 我发布的物品
  - 我下架的物品
  - 我收藏的物品
  - 我卖出的物品
  - 我买到的物品

### 后台功能

- **交易物品管理：**
  - 上线的二手物品（支持违规下架）
  - 下架的二手物品（支持永久删除）
- **订单信息管理：** 管理订单信息，支持删除操作
- **用户信息管理：**
  - 正常用户（可封号）
  - 违规用户（可以解封）
  - 管理员（可添加管理员）

## 系统角色

- 普通用户：可进行物品发布、查询、购买等操作
- 管理员：管理用户信息和交易物品，维护平台秩序
